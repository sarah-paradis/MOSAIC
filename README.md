# Modern Ocean Sediment Archive and Inventory of Carbon (MOSAIC)

Repository of Modern Ocean Sediment Archive and Inventory of Carbon (MOSAIC).

## Contribute to MOSAIC
MOSAIC is a collaborative database of organic carbon in marine sediments. If you have data (published and unpublished) and would like to contribute, download the [Excel template](https://github.com/sarah-paradis/MOSAIC/raw/main/Excel_templates/MOSAIC_input_excel_file_12-21.xlsx) and send it back to mosaic@erdw.ethz.ch
